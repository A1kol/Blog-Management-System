# Blog-Management-System
Простой и современный блог-менеджер на **JavaFX**, позволяющий создавать, редактировать, искать и сохранять посты.  

---

## Функции приложения

- **Просмотр постов**: Список постов отображается по заголовкам. Выбор поста показывает детали: заголовок, категорию и содержание.  
- **Добавление поста**: Нажмите `Add Post`, заполните поля (заголовок, категория, содержание) и сохраните.  
- **Редактирование поста**: Выберите пост и нажмите `Edit`. Поля откроются в редакторе, где можно изменить содержимое.  
- **Удаление поста**: Выберите пост и нажмите `Delete`.  
- **Поиск по заголовку**: Введите текст в поле поиска — список постов автоматически фильтруется.  
- **Сохранение в файл (.txt)**: Все посты можно сохранить вручную в текстовый файл `blog_posts.txt`. Формат удобен для чтения и печати.  
- **Загрузка постов из файла**: При запуске приложение автоматически загружает посты из `blog_posts.txt`, если файл существует.  
- **Современный интерфейс**: Чистый и минималистичный дизайн с современными кнопками и формами.  

---

## Используемые технологии

- **Java 17+** – современный язык программирования с поддержкой новых возможностей.  
- **JavaFX 17+** – для графического интерфейса.  
- **FXML** – разделение логики и интерфейса.  
- **ObservableList** – для динамического обновления списка постов.  
- **FXML Loader** – для динамического открытия редактора постов.  
- **Работа с файлами** – сохранение и загрузка постов в текстовом формате.
